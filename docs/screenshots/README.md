# Screenshots Guide

Place your app screenshots in this folder using these exact filenames:

- `splash-screen.png`
- `login-screen.png`
- `signup-screen.png`
- `home-screen.png`
- `add-expense-screen.png`
- `all-expenses-screen.png`
- `filter-screen.png`
- `chart-screen.png`
- `settings-screen.png`

## Recommended Steps

1. Run the app on a phone or emulator.
2. Open each screen one by one.
3. Capture a screenshot.
4. Crop the screenshot neatly if needed.
5. Save it in this folder with the matching filename.

## Example

If you capture the login screen, save it as:

`docs/screenshots/login-screen.png`

## Best Practice

- Use PNG format
- Keep all screenshots the same size if possible
- Use clear and uncluttered screens
- Avoid showing personal email, password, or private data
